# Question Bank

Offer these as starting points, then narrow based on the user's answers.

## Slide Ratio Options

- `LAYOUT_16x9`: safest default for modern screens and most business decks
- `LAYOUT_16x10`: slightly taller, useful for denser editorial slides
- `LAYOUT_WIDE`: strong visual impact for image-heavy decks
- `LAYOUT_4x3`: use only for legacy environments or explicit request

## Palette Families

Always present 3-4 options at a time. Each option must stay within 3-6 colors total.

- Executive Contrast: `1E2761`, `CADCFC`, `FFFFFF`, `7C93C3`
- Forest Editorial: `2C5F2D`, `97BC62`, `F5F5F5`, `D6E8C8`
- Terracotta Review: `B85042`, `E7E8D1`, `A7BEAE`, `5B3A29`
- Charcoal Signal: `36454F`, `F2F2F2`, `C44900`, `212121`
- Teal Data: `065A82`, `1C7293`, `9FD6E3`, `21295C`
- Cherry Premium: `990011`, `FCF6F5`, `2F3C7E`, `C14D5A`

## Typography Pairings

Prefer installed, PowerPoint-safe families.

- Georgia + Calibri: editorial but stable
- Arial Black + Arial: strong and simple
- Cambria + Calibri: classic corporate
- Trebuchet MS + Calibri: friendlier modern business
- Palatino + Garamond: refined, best for text-led decks
- Calibri + Calibri Light: neutral and efficient

## Layout Count Ranges

- 4 layouts: minimalist template, only essential deck structures
- 6 layouts: recommended baseline for a reusable business template
- 8 layouts: richer template with content variety
- 10 layouts: only for heavy reuse across multiple deck types

## Mandatory Layout Variants

For each mandatory layout, present 2-3 options.

### Title

- Full-bleed image with title overlay
- Clean title block with subtitle and small brand mark
- Dark cover with oversized title and compact kicker

### Agenda / Sommaire

- Numbered chapter list
- Two-column agenda cards
- Vertical timeline agenda

### Divider / Intercalaire

- Full-color divider with chapter number
- Minimal divider with large keyword
- Photo-backed divider with tinted overlay

### Conclusion / A Retenir

- Three key takeaways cards
- Single statement + next steps
- Summary grid with KPI recap

## Recommended Optional Layouts

Offer only the families that match the chosen tone and density.

- Two-column text + image
- Three-card benefits / pillars
- KPI or stat callout
- Comparison / before-after
- Timeline / process flow
- Quote / testimonial
- Table / matrix
- Chart insight slide
- Team or profile grid
- Closing contact slide

## Shape Language

- Sharp rectangles: structured and corporate
- Rounded rectangles: softer and product-friendly
- Circles + pills: dynamic, good for icons and KPIs
- Full-bleed zones: premium and image-led

## Motif Options

- Left accent band
- Top navigation ribbon
- Footer strip with page number
- Floating cards with soft shadow
- Large numeric markers
- Icon badges in circles

## Metadata Options

- Page number only
- Page number + section label
- Small logo + page number
- Footer source / confidentiality line
